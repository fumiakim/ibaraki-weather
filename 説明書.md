# 説明書（ユーザーマニュアル）

雨雲ナウキャスト & 7日間天気予報ビューアー の使い方ガイド。

---

## 1. はじめに

このアプリは、気象庁が公開している雨雲ナウキャスト・天気予報・アメダス観測値を組み合わせて、4 地点（つくば／柳津／姫路／名古屋）の天気を一画面で確認できるビューアーです。

## 2. 起動方法

### 2.1 オンラインで使う（推奨）

ブラウザで以下にアクセスするだけです:

> **https://fumiakim.github.io/ibaraki-weather/**

インストール不要、ログイン不要。

### 2.2 ローカルで使う

PC に保存した `index.html` をダブルクリックして開きます。

```bash
# macOS で開く例
open "/Users/fumiaki_air/Desktop/アプリ制作/ibaraki-weather/index.html"
```

> ⚠ ローカルで開くとブラウザの設定によっては気象庁データが取得できない場合があります。その場合はオンライン版をご利用ください。

## 4. 操作方法

### 4.1 地点を切り替える

地図左上の **地点ボタン**（つくば／柳津／姫路／名古屋）をクリックすると:

- 地図がその地点へアニメーション付きで移動
- マーカーのポップアップが開く
- **右側の予報・気温グラフも自動で当該地点（県）のものに切り替わります**
- ヘッダーのタイトルも更新

> 地図上の **マーカーをクリック** しても同じ動作になります。

### 4.2 雨雲の動きを見る

地図下のコントロールバーで操作します。

| コントロール | 動作 |
| --- | --- |
| **▶ 再生** ボタン | 過去1時間 → 現在 → 1時間先までの雨雲を 0.5 秒ごとに自動再生（ループ） |
| **❚❚ 停止** ボタン（再生中に表示） | 再生を止める |
| **スライダー** | 任意の時刻にジャンプ |
| **時刻ラベル** | 表示中フレームの月日・時刻 |
| **「実況」/「予報」バッジ** | 青= 観測実況／赤= 予報 を示す |
| **↻ ボタン** | 雨雲・予報・気温をすべて即時更新 |

雨雲は、強度に応じて青→緑→黄→橙→赤→紫 と色が変わります。地図左下の凡例を参照してください。

### 4.3 7日間予報の見方

各日のカードに表示される情報:

| 表示位置 | 内容 |
| --- | --- |
| 左 | 月/日（曜日。土曜は青、日曜は赤）|
| アイコン | 気象庁公式の天気アイコン＋天気名 |
| 中央 | 天気詳細文（3日先まで）。降水確率を 0–6 / 6–12 / 12–18 / 18–24 時の 4 区分で表示。4 日目以降は日次降水確率＋信頼度（A〜E） |
| 右 | 最高気温（赤）／最低気温（青）。週間予報は予測幅も併記（カッコ内）|

### 4.4 細分エリアタブ

予報パネル中央の **タブ**（例: 北部 / 南部）で、選択中の県内をさらに細かく切り替えできます。地点を切り替えると、その地点に最も近い細分が自動で選択されます。

### 4.5 当日の気温グラフ

予報パネル上部に、選択地点の最寄りアメダス観測点の **当日 0 時から現在までの 10 分値** を折れ線で表示します。

- グラフ内の点を **マウスオーバー** すると、その時刻と気温が表示されます。
- 右上の赤い数字は **最新値（小数1桁）と観測時刻**。
- 観測点名はグラフ上部に表示されます（例: 「今日の気温（観測点: つくば）」）。

## 5. 自動更新

操作不要で、以下の頻度で裏側で自動更新されます:

| データ | 更新頻度 |
| --- | --- |
| 雨雲ナウキャスト | 5 分ごと |
| 当日の気温（アメダス） | 10 分ごと |
| 7 日間予報 | 30 分ごと |

すぐ最新を取得したい場合は **↻ ボタン** をクリックしてください。

## 6. よくある質問

### Q1. 雨雲を拡大しすぎたらモザイクっぽくなった

雨雲ナウキャストは元データの解像度がズーム 10 までです。それ以上ズームインするとブラウザ側で拡大表示されます。元の解像度で見たいときはズームを下げてください。

### Q2. 予報が表示されない／古いまま

ネットワーク状況や気象庁側の一時的な障害が考えられます。↻ ボタンで再取得してみてください。それでも改善しない場合はブラウザコンソール（F12 / Cmd+Opt+I）でエラーを確認してください。

### Q3. 気温データが「取得できませんでした」と表示される

選択地点の近隣に気温観測のあるアメダスがない場合に発生します。ごく稀ですが、観測点側の保守等で当日全データが空のケースもあります。

### Q4. 表示地点を増やしたい・座標を変えたい

`index.html` の `LOCATIONS` 配列を編集すれば追加・変更できます。1 件あたりに必要な項目:

```javascript
{
  name: '表示名',         // ボタン・ヘッダーに使用
  lat: 緯度,
  lon: 経度,
  zoom: 12,              // 13 推奨
  note: 'ポップアップ補足',
  office: 'XXXXXX',      // 該当県の JMA office code
  prefName: '○○県',
  subAreaMatch: '南部',   // 一次細分名の部分一致文字列
  tempPointMatch: '○○',  // 週間気温観測点名の部分一致文字列
}
```

JMA office code は [気象庁の area.json](https://www.jma.go.jp/bosai/common/const/area.json) で確認できます。

### Q5. スマートフォンでも使える？

レスポンシブ対応しており、横幅 900px 未満では地図が上、予報が下の縦並びレイアウトに自動で切り替わります。ただし地点ボタンとマップコントロールはタッチ操作前提で配置していないため、操作性は PC が最適です。

### Q6. データの再配布や商用利用はできる？

気象庁および国土地理院のデータ利用規約に従ってください。本ビューアーは個人用途・学習目的の試作品です。

## 7. トラブルシューティング

| 症状 | 対処 |
| --- | --- |
| 画面が真っ白 | ブラウザを最新版に更新。または別ブラウザで開く |
| 地図は出るが雨雲が出ない | ↻ ボタン押下。または `https://www.jma.go.jp/bosai/jmatile/data/nowc/targetTimes_N1.json` にブラウザで直接アクセスして JSON が返ることを確認 |
| 予報パネルにエラー表示 | 気象庁側の障害の可能性。1〜2 分後に再試行 |
| 気温グラフが空 | 同上。深夜 0 時直後はその日の観測値がまだ無いため数十分間「データなし」になり得る |
